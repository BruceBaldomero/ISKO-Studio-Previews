# Moon Coffee & Brunch — Design System

**Business:** Moon — Speciality Coffee, Brunch & Bakes
**Address:** 249 Ewell Road, Surbiton, London KT6 7AA
**Instagram:** @moon.brunch.and.coffee
**Scope:** Outbound preview (one page)

---

## Vibe in one line
Neighbourhood gem — warm, unfussy, red-and-orange energy, food and coffee first.

## Palette
| Token | Hex | Usage |
|---|---|---|
| `--red` | `#C8312A` | Primary brand (shopfront, dominant UI) |
| `--red-dark` | `#A8261F` | Hover states, depth |
| `--orange` | `#D4601E` | Logo circle, accent, decorative elements |
| `--cream` | `#FDF8F3` | Page background |
| `--cream-2` | `#F5EDE3` | Section backgrounds, image placeholders |
| `--dark` | `#1C1412` | Body text, footer (warm near-black) |
| `--muted` | `#7A6965` | Secondary text, labels |

## Fonts
- **Display/headings:** DM Serif Display (warm, food-editorial, a little Italian café)
- **Body:** DM Sans (clean, friendly, unpretentious)
- *Deliberately avoids house default Fraunces + Inter Tight*

## Hero style
Split-screen: bold red left panel with large DM Serif Display heading + tagline + CTAs. Right panel = placeholder image. Decorative circle motifs reference the circular moon logo in CSS only (no image needed). Hero does NOT use a centred banner layout.

## Layout order
1. Preview banner (ISKO Studio watermark)
2. Hero — split panel
3. About — warm copy, warm cream background
4. What we do — 3 cards on red background
5. Hours & Find us — table + embedded Google Map
6. Footer — dark, minimal

## Notes
- Dog-friendly café — can mention in copy if helpful
- Food photography is the brand's strongest asset — placeholder slots are clearly labelled
- "Moon." with a full stop is how the brand feels — confident, one word
